# Clojure examples

- Find `nth` element in a collection

```clojure
(defn nth [arr n]
  ((comp last (partial take (inc n))) arr))
```

Using recursion approach. eg

```clojure
(defn nth [arr n]
  (if (= n 0)
    (first arr)
    (nth (rest arr) (- n 1))))
```

Lazy Fibonacci sequence

```clojure
(defn fib [n]
  (take n (map first
               (iterate
                (fn [[a b]] [b (+ a b)])
                [1 1]))))
```
